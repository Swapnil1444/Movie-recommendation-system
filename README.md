# Movie Recommendation Chatbot

A Java web application built with Spring Boot that provides movie recommendations via a simple chatbot interface.

## Features

- Web-based chat interface
- Basic movie recommendation logic
- Built with Spring Boot and Thymeleaf

## Getting Started

### Prerequisites

- Java 17
- Maven

### Running the Application

1. Clone the repository
2. Navigate to the project directory
3. Run `mvn spring-boot:run`
4. Open http://localhost:8080 in your browser

## Usage

Enter a genre or movie name to get a recommendation.
